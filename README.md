🚀 CRM-CORE Clone

A lightweight clone of CRM-CORE, built with a focus on simplicity, speed, and cost efficiency.

🎯 Goal

Reproduce a well-structured vanilla stack that is:

⚡ Fast
🪶 Lightweight
💸 Cheap to run
🧱 Easy to maintain

No unnecessary abstractions. No bloated frameworks. Just clean architecture and predictable behavior.

🧠 Philosophy

This project follows a pragmatic approach:

Keep logic explicit, not hidden behind magic
Prefer vanilla implementations over heavy dependencies
Optimize for real-world performance, not theoretical elegance
Build something that can scale when needed, not before
🛠️ Stack
🐘 PHP (backend)
🧩 Vanilla JavaScript (frontend)
🗄️ MySQL / MariaDB
🌐 Nginx

No frameworks. No ceremony. Just control.
